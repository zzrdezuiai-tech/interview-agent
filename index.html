从一次面试，到一套可持续训练系统

面鸭Agent 面向产品经理、AI 产品经理、增长运营、数据分析师和技术转产品候选人，提供多岗位、多风格、多语言的 AI 面试训练。

简历 / JD 解析

岗位画像与能力矩阵

解析候选人简历、目标 JD、岗位级别和行业背景，生成面试重点、风险点和能力 Rubric。

多轮动态追问

AI 面试官实时追问

支持严厉型、友善型、压力型、Case 型、技术型和业务型面试官，根据回答动态调整问题。

报告 / 复盘

结构化面后报告

输出逐题评分、能力雷达、逻辑漏洞、表达建议、可复用回答模板和下一轮训练计划。

Agent Harness 架构

系统通过 Planner、Question、Evaluator、Coach、Report 与 TTS 模块协作，验证 MiMo 在长链路 Agent、结构化评估和语音交互中的表现。

Planner Agent

根据简历、JD 和岗位目标生成面试路径，决定能力维度、问题顺序和难度曲线。

Question Agent

生成分阶段问题与动态追问，支持中文、英文和中英混合面试。

Evaluator Agent

基于结构化 Rubric 输出评分、理由、风险标签和改进建议。

Coach Agent

将评分转化为可执行训练建议，包括 STAR 模板、数据分析路径和表达优化。

Report Agent

生成面后复盘报告、历史训练对比和下一阶段训练计划。

MiMo TTS

将面试官问题和追问语音化，构建更接近真实面试的交互体验。

完整训练流程

从输入材料到面后复盘，所有模块围绕真实求职训练场景设计。

1输入材料上传简历、目标 JD、岗位级别、目标公司与语言偏好。
2生成画像构建候选人 Profile、岗位能力矩阵和高风险问题。
3多轮面试10–15 轮动态追问，支持多种面试官风格。
4实时评分按 Rubric 输出结构化评分、理由和下一轮策略。
5复盘提升生成报告、回答模板、薄弱项分析和二次训练计划。
