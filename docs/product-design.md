# 产品设计文档

## 产品定位

面鸭 interview agent 是面向求职训练的多模态 AI 面试 Agent 平台，帮助候选人在接近真实面试的环境中进行结构化训练。

## 核心价值

- 将简历、JD、岗位能力矩阵、多轮追问、实时评分、TTS 语音播报和面后报告整合为完整闭环
- 将面试训练从随机问答升级为可规划、可评分、可复盘、可持续提升的训练系统
- 支持不同岗位、不同面试官风格和不同语言场景

## 用户路径

1. 用户输入简历、目标 JD、岗位级别和目标公司。
2. 系统生成候选人画像和岗位能力矩阵。
3. Planner Agent 规划面试路径。
4. Question Agent 进行多轮提问和追问。
5. Evaluator Agent 实时评分并标记风险点。
6. Coach Agent 给出回答优化建议。
7. Report Agent 输出面后复盘报告。
8. 用户根据训练计划进行下一轮模拟。

## 首批支持岗位

- 产品经理
- AI 产品经理
- 增长运营
- 数据分析师
- 技术转产品候选人

## 页面模块

- Hero 区：项目定位、核心描述和项目进展入口
- 产品能力区：简历 / JD 解析、多轮追问、面后报告
- 架构区：Agent Harness 模块说明
- 流程区：完整训练流程
- 更新日志区：项目进展记录
- 联系弹窗：邮箱和 GitHub 链接
